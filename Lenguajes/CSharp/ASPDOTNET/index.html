Proyecto realizado en quinto semestre


Se fragmenta una base de datos y se replica parte de ella en otra computadora y por medio del IIS de windows server 2012 R2 se levanta un servidor de forma local en donde podemos acceder a la página. Se guarda una transparencia respecto al usuario y también se distingue los fragmentos de la base de datos.
Temas: IIS, DNS, Fragmentación, Replicación, ASP.NET
