Proyecto realizado en quinto semestre


Por medio de estructuras de datos se genera un arbol con varios movimientos posibles y se elije el mejor. En la materia de IA se comprobó que este método es de los más ineficientes para resolver este tipo de problemas y además la heurística debe considerar distancias no solo en 2D sino en 3D también.
Temas: estructura de datos, heuristica, distancia de manhatan, IA
