Proyecto realizado en cuarto semestre por: Chicaiza Carlos , Díaz Danny (Magody) y Vinueza Rafael


Desarrollamos un pequeño minijuego usando solo JAVA como lenguaje para entender los fundamentos de un juego 2D. En el juego se puede destruir enemigos, recolectar dinero y usarlo ya sea en inversiones o una tienda, y usar magia.
arriba = W
abajo = S
izquierda = A
derecha = D
salir = ESCAPE
correr = SHIFT
poderBola de fuego = 1
poder_tiempo = T
disparar_arma = E
recargar_arma = R
ataque_espada = 2
Temas: estructura de datos, Graphics, Nativo


Haga clic en el boton de la derecha para ver la documentación completa.
