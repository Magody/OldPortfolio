Proyecto realizado en quinto semestre


Usando un dataset proporcionado por Kaggle de los ratings de videos de youtube utilizamos un cromosoma de N genes, donde N es la cantidad de recomendaciones que se va a hacer, para elegir el mejor grupo de recomendaciones para una combinación de temas. La precisión al forzar que cada gen sea parte de la recomendación, es algo baja. El enfoque fue hecho así apropósito por métodos didácticos.
Temas: algoritmos genéticos, función de aptitud, YouTube


Haga clic en el boton de la derecha para ver el dataset utilizado en formato .CSV. Pesa 59.8MB
