Proyecto realizado en cuarto semestre por: Borja Grace, Burbano Renato y Díaz Danny (Magody)


Backtracking es una técnica de fuerza bruta optimizada. Para resolver este puzzle siempre encontramos una primera solución de una forma muy rápida, pero buscamos la mejor por lo que poco a poco y mediante cotas vamos optimizando las posibilidades.
Temas: estructura de datos, heuristica, backtracking, IA, búsqueda A*
